# vue-mall

> vue-resource+vue-router+mint-ui+mui搭建的单页面商城

## 使用

1. 装包 `npm i`
2. 运行 `npm run dev`
3. 编译 `npm run build`
4. 推荐使用`cnpm i`命令装包，速度比`npm`快
5. 安装`cnpm`方法: `npm i -g cnpm`。安装后用`cnpm`替换`npm`
